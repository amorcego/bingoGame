Crie seu banco de dados e execute o arquivo SQL "criarTabelas.sql" para criar as tabelas nescessárias, 
fique a vontade para fazer as modificações que desejar.
   Lembrando que todo usuário logado no sistema é adicionado à tabela "bingo" e historico de jogos e jogadores são adicionados à tabela "jogos", então será nescessário adicionar um script para apagar usuários que não estiver online, caso contrário voçe sempre terá que entrar com um nome de usuario diferente e seu DB ficará sobrecarregado de dados.
